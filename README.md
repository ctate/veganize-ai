## Quickstart

1. Add OpenAI API to a new file called `.env.local`:

    ```sh
    OPENAI_API_KEY=your_key_goes_here
    ```

1. Install dependencies:

    ```sh
    npm install
    ```

1. Run development server:

    ```sh
    npm run dev
    ```

1. Go to http://localhost:3000
